## 🔧 集成指南

### 快速开始
```python
from framework.shared.event_bus.event_bus import EventBus

bus = EventBus(member_name="your_member")
bus.subscribe("event_type", your_handler)
bus.publish("event_type", {"data": "value"})
bus.process_events()
```
